# React + TypeScript + Vite
启动
```
npm install
npm run dev
```
打包
```
npm run build
```
## 技术栈
- React
- react-router-dom
- TypeScript
- scss
- tailwindcss
- axios
- Vite
- antd
- react-redux
- @reduxjs/toolkit  (react-redux工具插件)
- classnames (class类工具)
- react-i18next i18next (国际化)
- nprogress (进度条插件)
